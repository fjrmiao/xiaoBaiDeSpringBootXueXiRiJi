# 官方文档是个好东西
https://router.vuejs.org/zh/installation.html

# 安装
```
npm install vue-router
```
# 使用
```
import Vue from 'vue'
import VueRouter from 'vue-router'

Vue.use(VueRouter)
```

# 起始页（默认是app.vue）里要有router-view
```
  <div>
    <router-view />
  </div>
```

# 通过scoped限定css的适用范围
```
<!-- Add "scoped" attribute to limit CSS to this component only -->
<style scoped>
```
