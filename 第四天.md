# 试一下Vue
一个基本的例程
```
<!DOCTYPE html>
<html>
<head>
	<title></title>
</head>
<body>
	<div id="demoapp">
  {{ message }}
</div>



<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
<script type="text/javascript">
	var app = new Vue({
  el: '#demoapp',
  data: {
    message: 'example'
  }
})

</script>
</body>
</html>
```

两个变量的例程
```
<!DOCTYPE html>
<html>
<head>
	<title></title>
</head>
<body>
	<div  id="demoapp">
	<h2>{{title}}</h2>
	<div>{{ message }}</div>
	</div>



<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
<script type="text/javascript">
	var app = new Vue({
  el: '#demoapp',
  data: {
  	title: '名言',
    message: '好人一生平安'
  }
})

</script>
</body>
</html>
```
